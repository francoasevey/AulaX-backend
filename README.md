# AULA X - Backend API

Backend de la plataforma educativa AULA X desarrollado con FastAPI.

## Tecnologías

- Python 3.11+
- FastAPI
- MongoDB + Motor
- JWT Authentication
- OpenAI API

## Instalación

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Endpoints

- http://localhost:8000/docs - Swagger UI
- http://localhost:8000/redoc - ReDoc
